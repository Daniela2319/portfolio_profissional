Projeto Site Rota do Pão

Descrição:
O sistema foi idealizado para atender à demanda de consumidores que buscam praticidade na aquisição de pães e outros itens essenciais, com a possibilidade de configurar entregas recorrentes diretamente no conforto de suas casas.

Principais funcionalidades:
- Modelo de conveniência: Compra rápida e fácil, com uma interface intuitiva.
- Modelo de recorrência: Configuração de entregas automáticas e periódicas para produtos selecionados.
- Catálogo de produtos: Exibição organizada de produtos com detalhes e preços.
- Carrinho de compras: Gerenciamento de itens selecionados para compra.
- Checkout simples: Processo de finalização de compras eficiente.

Tecnologias utilizadas:
- Visual Studio Code: Ambiente de desenvolvimento integrado (IDE).
- HTML: Estrutura e organização do conteúdo da aplicação.
- CSS: Estilização e criação de design responsivo.
- JavaScript: Dinamismo e interatividade no frontend.
- Git: Controle de versão para organização e histórico de mudanças.
- GitHub: Plataforma para hospedagem e colaboração do código.

Projeto Blog Colaborativo

Descrição:
O Blog Colaborativo é um projeto desenvolvido em grupo, utilizando Python e o framework Django, com o objetivo de criar uma plataforma completa para publicações e interação entre usuários. O sistema oferece recursos robustos de postagem, comentários, autenticação, gerenciamento de perfis de usuários e uma funcionalidade eficiente de busca por postagens.

Principais funcionalidades:
- Sistema de Postagens: Criação, edição e exibição de posts organizados em ordem cronológica.
- Comentários: Permite que os usuários interajam com as postagens, deixando seus feedbacks diretamente na página de detalhes.
- Autenticação de Login: Sistema seguro de registro e login, garantindo o acesso exclusivo às funcionalidades personalizadas.
- Gerenciamento de Perfil de Usuário: Cada usuário pode personalizar sua bio e imagem de perfil.
- Busca de Postagens: Pesquisa eficiente de conteúdos por palavras-chave, otimizando a experiência do usuário.

Tecnologias utilizadas:
- Python: Linguagem de programação principal para o desenvolvimento do backend.
- Django: Framework para construção de aplicações web escaláveis e seguras.
- HTML5: Estruturação das páginas do blog.
- CSS3: Estilização das interfaces com design responsivo.
- Bootstrap: Framework para facilitar o design responsivo e estilização.
- Banco de Dado: PostgreSQL.
- Cloud: AWS.
- Git: Controle de versão do código.
- GitHub: Plataforma para hospedagem e colaboração do código.
